# Zyro Bandung Tech - Project Management System

[![Laravel](https://img.shields.io/badge/Laravel-11.x-red.svg)](https://laravel.com)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**Zyro Bandung Tech** adalah aplikasi berbasis web yang dirancang untuk mengelola proyek, klien, dan sistem pembayaran secara terpusat. Aplikasi ini membantu tim teknis dalam memantau progress pekerjaan dan administrasi klien dengan lebih terstruktur.

## 🌟 Fitur Utama

* **Management Project:** Pantau status dan detail setiap proyek yang sedang berjalan.
* **Client Database:** Pengelolaan data klien yang terintegrasi.
* **Team Collaboration:** Pengaturan tim untuk setiap project yang dikerjakan.
* **Payment Monitoring:** Pencatatan dan pemantauan status pembayaran dari klien.
* **Dashboard:** Ringkasan aktivitas dan metrik penting perusahaan.

## 🛠 Tech Stack

* **Framework:** Laravel 11
* **Database:** MySQL / MariaDB
* **Frontend:** Blade Templating & Tailwind CSS
* **Language:** PHP 8.2+
